Method __builtin.Nettle.MAC()->iv_size()

Method iv_size

int(0..) iv_size()

Description

Returns the size of the iv/nonce (if any).

Some MACs like eg Crypto.SHA1.HMAC have fixed ivs, in which case this function will return 0.
