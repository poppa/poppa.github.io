Method __builtin.Sql.Connection()->streaming_query()

Method streaming_query

int|.Result streaming_query(object|string q)

Description

Send an SQL query synchronously to the SQL-server and return the results streaming in untyped mode.

For the arguments, please see the low_big_query() function.

The result is returned as a streaming Sql.sql_result object in untyped mode. This allows for having results larger than the available memory, and returning some more info about the result. Returns 0 if the query didn't return any result (e.g. INSERT or similar). For the other arguments, they are the same as for the query() function.

Called by streaming_query() after it has processed any bindings.

This variant function is intended for overriding by drivers.

Note

Streaming operation is not supported by all sql databases. The default implementation falls back to calling big_query().

See also

big_query, streaming_typed_query
